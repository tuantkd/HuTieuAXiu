HUONG DAN ADMIN POS A XIU

1. Mo XAMPP va bat `Apache` + `MySQL`.
2. Tao database `so_ban_hang_a_xiu`.
3. Import file `database.sql`.
4. Neu database cu da tung dung schema admin rieng, import them `admin/setup.sql`.
5. Mo trinh duyet:
   `http://localhost/so-ban-hang-a-xiu-full/admin/login.php`

Tai khoan mau:
- `admin / 123456`
- `nhanvien / 123456`

Phan quyen:
- `admin`: xem va quan ly toan bo man hinh.
- `nhanvien`: chi truy cap duoc `pos.php`.

Neu can tao lai tai khoan mau:
- Chay `http://localhost/so-ban-hang-a-xiu-full/admin/create_users.php`
